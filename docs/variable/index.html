    public IFCObjectDefaults: {String: Object} source 

    import {IFCObjectDefaults} from 'xeokit-sdk/src/viewer/metadata/IFCObjectDefaults.js'

    Default initial properties for Entitys loaded from models accompanied by metadata.

    When loading a Node from a model, a loader plugin such as GLTFLoaderPlugin can optionally also load metadata to create a MetaModel corresponding to the model Node.

    The Node willl represent the model, and have a tree of sub-Nodes and Meshes that represent the objects within the model.

    The MetaModel will contain a MetaObject for each of the object Nodes and Meshes.

    This maps each MetaModel#type to a set properties to Entity property values to MetaObject types

    This is used within GLTFLoaderPlugin, GLTFBigModelLoaderPlugin, BIMServerLoaderPlugin and BIMServerBigLoaderPlugin.
