    public get abstract isStateSortable: boolean source

    Property with final value true to indicate that xeokit should render this Drawable in sorted order, relative to other Drawable of the same class.

    The sort order is determined by Drawable#stateSortCompare.

    Sorting is essential for rendering performance, so that xeokit is able to avoid applying runs of the same state changes to the GPU, ie. can collapse them.

    public getPickResult(pickResult: PickResult) source

    Given a PickResult that contains a PickResult#primIndex, which indicates that a primitive was picked on the Drawable, then add more information to the PickResult about the picked position on the surface of the Drawable.

    Architecturally, this delegates collection of that Drawable-specific info to the Drawable, allowing it to provide whatever info it's able to.

    Params:

    NameTypeAttributeDescription
    pickResult PickResult

    The PickResult to augment with pick intersection information specific to this Mesh.
    pickResult.primIndex *
    • optional

    Index of the primitive that was picked on this Mesh. Essential for obtaining the intersection information.
    pickResult.canvasPos *
    • optional

    Canvas coordinates, provided when picking through the Canvas.
    pickResult.origin *
    • optional

    World-space 3D ray origin, when ray picking.
    pickResult.direction *
    • optional

    World-space 3D ray direction, provided when ray picking.
