    Viewer

    The WebGL-based 3D Viewer class at the heart of the xeokit SDK.

    • A Viewer wraps a single Scene
    • Add Plugins to a Viewer to extend its functionality.
    • Viewer#metaScene holds metadata about Models in the Viewer's Scene. Load and unload metadata using Viewer#createMetadata and Viewer#destroyMetadata.
    • Save and load the state of a Viewer as JSON with Viewer#getBookmark and Viewer#setBookmark. Installed Plugin instances will also save and load their state to and from the JSON.
    • Use Viewer#cameraFlight to fly or jump the Scene's Camera to target positions, boundaries or Nodes.

    {Number} The number of times the Viewer#scene renders per frame.

    {Boolean} When doing multiple passes per frame, specifies if to clear the canvas before each pass (true) or just before the first pass (false).
