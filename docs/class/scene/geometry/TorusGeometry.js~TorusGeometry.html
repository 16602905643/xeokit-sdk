    TorusGeometry

    Extends:

    Component → Geometry → TorusGeometry

    Defines a toroidal shape for one or more Meshes.

    Usage

    Creating a Mesh with a TorusGeometry and a PhongMaterial with diffuse Texture:

    new Mesh(myViewer.scene, {
    geometry: new TorusGeometry(myViewer.scene, {
         center: [0,0,0],
         radius: 1.0,
         tube: 0.5,
         radialSegments: 32,
         tubeSegments: 24,
         arc: Math.PI * 2.0
     }),
     material: new PhongMaterial(myViewer.scene, {
         diffuseMap: new Texture(myViewer.scene, {
             src: "textures/diffuse/uvGrid2.jpg"
         })
     })
 });
